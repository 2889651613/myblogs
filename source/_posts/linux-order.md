### 
1. 基本命令
    1. 查看当前目录

     ``` 
        pwd
     ```
    2. 撤换目录命令
    
    ```
       cd 
    ```
    3. 查看当前目录内容
     ```
      ls
     ```

2. 系统命令
    1. 查看活动进程
    ```
       top
    ```
    2. 跟踪某个程序
    ```
       trace
    ```
3. 网络命令
   ```
     curl 
   ```
